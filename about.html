About CISA KEV / BOD 22-01 Deadline Planner

CISA KEV Deadline Planner is a free browser worksheet for matching pasted CVEs against a bundled CISA Known Exploited Vulnerabilities catalog snapshot and exporting a BOD 22-01 remediation due-date planning queue.

BOD 22-01 Caveat

The bundled catalog includes CISA required action and due-date fields. Use them as planning inputs, then verify whether BOD 22-01 applies to the organization, system, and remediation path before treating any date as a compliance obligation.

Boundaries

The tool runs entirely in the browser with a bundled local catalog file. It has no signup, cookies, analytics, beacons, uploads, browser storage, or external scripts.

This is an unofficial informational worksheet. It is not CISA affiliation, legal advice, security advice, remediation advice, BOD 22-01 compliance advice, a scanner, vulnerability validation, or environment-specific risk analysis.
